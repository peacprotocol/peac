# @peac/mappings-a2a

Agent-to-Agent Protocol (A2A) integration for PEAC.

Maps PEAC evidence carriers to A2A metadata (TaskStatus, Message, Artifact) per A2A spec v0.3.0.

## Installation

```bash
pnpm add @peac/mappings-a2a
```

## Documentation

See [peacprotocol.org](https://www.peacprotocol.org) for full documentation.
