# PEAC Integration Kit: Content Signals

Integration guide for adding PEAC receipts to Content Signals (C2PA adjacent) workflows.

## Status

Integration kit: expanding to full guide. The `@peac/mappings-content-signals` package is published and stable since v0.11.2. AIPREF vocab draft-05 version pinning shipped in v0.12.3.

## Quick Start

```bash
npm install @peac/mappings-content-signals
```

See [examples/content-signals](../../examples/content-signals/) for a working content signals observation example.

## Reference

- `@peac/mappings-content-signals`: robots.txt, AIPREF, tdmrep.json parser and signal mapper
- Content Signals: AI content provenance and use-policy signaling
